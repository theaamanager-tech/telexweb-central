# TelexWeb Central 🏪

**Single source of truth** untuk stok credential digital. Hubungkan banyak toko (Telegram, Website) ke 1 Central Stock.

## 📦 Isi Repo

| Folder | Isi |
|--------|-----|
| `central/` | Panel + API (Next.js, deploy Vercel) |
| `database/` | SQL schema untuk Supabase |
| `guides/` | Panduan setup super pemula |
| `stores/web/` | Clone webtestgum → diedit (coming soon) |
| `stores/telegram/` | Clone telegramstore → diedit (coming soon) |

## 🚀 Quick Start

1. Baca `guides/01-persiapan-dan-setup-lengkap.md`
2. Setup Supabase Central → paste `database/01-central-schema.sql`
3. Deploy `central/` ke Vercel
4. Masuk panel → tambah produk + variant (SKU)
5. Tambah stok → toko mulai bisa claim
